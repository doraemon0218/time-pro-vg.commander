# Time Pro VG Commander

Time Pro VG の勤怠打刻修正申請をワンクリックで一括処理するWebアプリ。

## 機能

- **月間カレンダーUI**: 日付ごとの勤務ステータスを一覧表示・編集
- **曜日別デフォルト設定**: 月〜金 7:30〜16:30 などを設定すると自動でスケジュール生成
- **ステータス管理**: 通常出勤 / 有給休暇 / 出張 / 非勤務日 / 祝日
- **日本の祝日を自動判定**
- **一括修正申請**: Playwright でTime Pro VGに自動ログイン・申請
- **データ永続化**: スケジュールはローカルJSONに保存

## セットアップ

```bash
# 依存関係インストール
npm install

# Playwrightブラウザをインストール
npx playwright install chromium

# 開発サーバー起動
npm run dev
```

→ http://localhost:3000 を開く

## 設定手順

1. 右上の ⚙️ ボタンをクリック
2. **Time Pro VG の URL・ユーザー名・パスワード** を入力
3. 曜日別の出勤時刻デフォルトを設定（例: 月〜金 7:30〜16:30）
4. 有給・出張の日はカレンダーの日付をタップして変更
5. **🚀 修正申請** ボタンで一括申請

## Time Pro VG セレクタの調整

`src/lib/automation.ts` の `SELECTORS` を実際の画面に合わせて調整してください。
ブラウザの開発者ツール → 要素を右クリック → 「セレクタをコピー」で取得できます。

## 院内WiFi環境での利用

このアプリはローカルサーバーとして動作します。
病院のPC（またはサーバー）で `npm run dev` を起動し、
同一WiFi内の端末から `http://[PCのIPアドレス]:3000` でアクセスできます。
